# fulfillment-risk-predictor
Predicts fulfillment exceptions using operational warehouse features and exposes predictions through a lightweight ML service.

Business problem: missed service levels, delays, or routing exceptions hurt labor efficiency and customer experience
ML goal: identify high-risk units before failure
Prediction target: binary classification (risk_flag = 0/1)
Relevance: This use case is key in alerting stakeholders to risk levels and aiding in decision making
